## 介绍
Loon自用规则，规则、插件来自网站，本人不作维护。只进行了整合并整理我适合个人使用的Loon规则

## 功能
- 自动分流、测速
- 负载均衡、广告过滤

## 使用
- 复制 **[规则链接](https://raw.githubusercontent.com/dqzboy/Loon_Script/main/conf/loon.conf)** 导入Loon
